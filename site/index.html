Барьер

Руководство пользователя

О расширении

Данное расширение является попыткой помочь людям бороться с зависимостями в интернете. В этот список входят зависимости от "взрослого" контента, социальных сетей, игровых сервисов и т.п.(на данный момент только взрослый контент).

Основное отличие Барьер от похожего софта - плагин не блокирует доступ к сайту полностью. Он даёт пользователю выбор, либо остаться, либо уйти и помогает сделать правильный выбор.
Каким образом плагин помогает принять верное решение? Всё просто - пользователю будут показаны слова, которые могут дать мотивацию покинуть сайт.

Использование

Прежде чем пользоваться плагином, его надо настроить, заполнив список сайтов, на которых должны быть показаны предупреждения.

Для этого надо перейти в настройки плагина. Для этого выберите Настройки и управление Google Chrome -> Дополнительные инструменты -> Расширения.
Найдите в списке Барьер и нажмите Параметры. Откроется страница настроек плагина.

Рис.1 Настройки

Для добавления нового сайта надо нажать Добавить сайт и ввести в поле ввода адрес. В выпадающем списке выбираем тип сайта (на данный момент только adult). Далее нажимаем сохранить.
Теперь на сайтах домена example.com будет показано предупреждение и цитаты для категории adult.

Рис.2 Предупреждение

Нажимая на зеленого ангелочка пользователя перекидывает на google.com, нажимая на красный череп пользователь закрывает предупреждение.
